# PhotoMapper
A plug-in for QGIS for
- Plotting points(lat,long) on QGIS
- Plotting a graph on QGIS: input comes in 2 files: vertex_file and edge_list
+ vertex_file: id[tab]coordinates
+ edge_list: id_from[tab]id_to
